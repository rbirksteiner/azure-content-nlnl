# 搭配 Azure Multi-factor Authentication 使用驗證碼登入行動應用程式


下列資訊將說明當您搭配驗證碼使用 Multi-Factor Authentication 時，將 Multi-Factor Authentication 使用於您的行動應用程式的經驗。

## 搭配行動應用程式使用驗證碼進行登入

<ol>

<li>使用您的使用者名稱和密碼登入應用程式或服務，例如 Office 365。</li>
<li>Microsoft 將會提示您輸入驗證碼。</li>


<center>![Setup](./media/multi-factor-authentication-end-user-signin-app-verify/verify.png)</center>

<li>在電話上開啟 Azure Authenticatior 應用程式，並登入時的方塊中輸入驗證碼。</li>

<center>![Setup](./media/multi-factor-authentication-end-user-signin-app-verify/phone.png)</center>


<li>您現在應已登入。</li>
